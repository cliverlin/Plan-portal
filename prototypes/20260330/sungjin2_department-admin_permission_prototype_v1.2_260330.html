전체관리자 조회·편집 권한 템플릿 목록 표시 읽기전용 미표시
전체관리자는 모든 그룹 정책 템플릿을 조회·편집할 수 있습니다. 읽기전용 배지가 표시되지 않으며 모든 카드에서 설정 변경이 가능합니다.
  • 전체 관리자는 회사 전체의 모든 템플릿을 볼 수 있음
  • 읽기전용 배지 없이 모든 카드가 편집 가능 상태
  • 즐겨찾기(★), 더보기(⋮) 메뉴 모두 정상 동작

조회 범위 제한 읽기전용 배지 편집 가능 조건 판정
부서범위 관리자(개발팀 담당)는 자신의 관리 범위와 관련된 템플릿만 목록에 노출됩니다. 아래 기준으로 읽기전용 배지가 표시됩니다.
  • 편집 가능: 내가 생성 + 대상 전원이 내 관리 범위 내 → 배지 없음
  • 읽기전용 ①: 타인이 생성한 템플릿에 내 관리 범위 내 대상 포함 → 읽기전용 표시
  • 읽기전용 ②: 내가 생성했지만 관리 범위 밖 대상이 포함 → 읽기전용 표시
  • 읽기전용 카드를 클릭하면 탭 3 (읽기전용 정책 설정 화면)으로 이동합니다
