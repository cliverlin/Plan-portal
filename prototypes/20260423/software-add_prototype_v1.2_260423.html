탐지 조건이란?

소프트웨어가 PC에 설치되어 있는지 판단하는 기준입니다.
아래 세 가지 방법 중 하나 이상을 입력해야 합니다.

레지스트리 경로 권장
설치 프로그램이 Windows 레지스트리에 기록한 경로입니다. MSI·NSIS 등 표준 방식으로 배포된 소프트웨어에서 사용할 수 있으며, 가장 신뢰할 수 있는 탐지 방법입니다.
확인 방법
실행(Win+R) → regedit 실행
HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Uninstall\ 하위 항목 참고
설치 경로 보조
소프트웨어가 설치된 폴더 경로를 기준으로 탐지합니다. 레지스트리에 등록되지 않는 portable 앱이나 비표준 설치 방식에 유용합니다.
주의
사용자 지정 경로에 설치된 경우 탐지에서 누락될 수 있습니다.
프로세스명 보조
소프트웨어 실행 시 생성되는 .exe 파일명으로 탐지합니다. 소프트웨어가 실행 중인 PC에서만 탐지됩니다. 설치되어 있어도 실행하지 않은 PC에서는 탐지되지 않습니다.
확인 방법
작업 관리자(Ctrl+Shift+Esc) → 프로세스 탭에서 실행 파일명 확인
OR 조건이란?
여러 탐지 기준을 등록하면, 그 중 하나라도 일치할 경우 해당 소프트웨어가 설치된 것으로 인식합니다. 설치 경로가 다를 수 있는 버전이나 portable 버전을 함께 탐지할 때 활용합니다.
